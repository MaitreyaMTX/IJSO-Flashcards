# 🧪 IJSO Master Flashcards

An interactive flashcard application for the International Junior Science Olympiad (IJSO).

## Features

- ✅ Biology, Chemistry, Physics, General Science
- ✅ Interactive flip cards
- ✅ Search system
- ✅ Subject filtering
- ✅ Progress saving
- ✅ Keyboard controls
- ✅ Offline support
- ✅ Installable as a PWA

## How to run locally

Download or clone the repository.

Open: index.html in you browser
